# VaaniX AI

Welcome to **VaaniX AI**, an advanced autonomous education operating system and AI platform.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Deployment

This project is configured for seamless deployment on [Vercel](https://vercel.com/). Any pushes to the `main` branch will automatically trigger a new deployment.
